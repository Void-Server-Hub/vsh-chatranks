VSH Chatranks

Overview

VSH Chatranks is a Minecraft Add-on that allows you to create custom chat ranks, mute players, show dimensions, and much more! This add-on is designed to work with Minecraft version 1.21.70 and above.

Features

  • Customizable customName: and chatName: tags.
  • Advanced features like chat ranks, mute options, and dimension control.
  • Fully compatible with Minecraft version 1.21.70.

Installation

  1. Download the add-on.
  2. Place the add-on file in your Minecraft "behavior_packs" directory.
  3. Enable the add-on in the world settings under Behavior Packs.
  4. Enjoy the features of VSH Chatranks!

===== Rank & Name Customization =====

 - Displays the normal chat rank next to the player's name in chat
 - Changes the player's name above their model
 - Changes the player's name in chat

===== Hiding Sections =====

 - Hides the player's name above their model
 - Hides ranks in chat
 - Hides ranks next to the player's name above their model
 - Hides the clan tag from name above player model
 - Hides the clan tag from a player in chat

===== Mute & Spam Control =====

 - Mutes the player and prevents them from sending messages in chat
 - Allows player to repeat messages
 - Allows player to message without waiting

===== Configuration Options =====

 - Displays the player's health in a 20/20 format below their name
 - Shows the dimension next to the player in chat

===== Clan Tag =====

 - The clan tag that appears before a name (Preferably 4-6 characters | Only supports 1 and oldest clantag applied)

Example:

/tag @s add rank:Cool
/tag playerName rank:Cool

Supported rank formats:

'rank:', 'r:', 'Rank:', 'role:', 'title:', 'R:'
